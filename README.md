# Students-Form
## Project Description

Student Forms is a simple C# Windows Forms application. It is used to collect student details through a form.
A simple  C# Windows Forms application for collecting student details like photo, name, date of birth, gender, qualification, experience and project information.


## Features

* Upload student photo
* Enter student name
* Select date of birth
* Choose gender
* Select marital status
* Choose qualification
* Select experience
* Enter project details
* Show all details in a message box

## Technologies Used

* C#
* Windows Forms
* Visual Studio

## How to Run

1. Open the project in Visual Studio.
2. Build the project.
3. Run the application.
4. Fill the form details.
5. Click the Submit button.

## Output

The application displays the entered student details in a message box.
